Build 1.0.0 (13)
April 20, 2026 iOS · TestFlight

A big quality-of-life release — smarter onboarding, polished UI throughout, and a round of targeted fixes based on early tester feedback.

Latest
✨4 Enhancements
⚙️8 UI Improvements
🐛7 Bug Fixes
🔧1 Technical
✨ Enhancements
  • You can now add nitrogen content when entering products during onboarding — helping the app give you more accurate lawn care recommendations from day one.
  • Added a quick link on the application screen so you can add new products on the fly if you can't find what you're looking for without losing your place.
  • The Weather Details page has been cleaned up — unnecessary icons removed for a cleaner, more focused view.
  • Activity logging now supports multiple photos per entry, and a dedicated "Take Photo" button makes capturing your lawn's progress even easier.
⚙️ UI Improvements
  • The AI assistant (Greenkeeper) has been renamed — cleaner branding throughout the app.
  • The login screen now stays out of your way — you'll only see it when you actually need it.
  • AI chat responses no longer include distracting hashtags — conversations feel cleaner and more natural.
  • Password login is now the default on the login screen, making it faster to get into the app.
  • You can now set or update your password from your Profile — useful if you signed up via Google and want a password login option too.
  • The Growing Degree Days (GDD) card on the home screen has been streamlined — now shows just the two key numbers with a more compact layout.
  • The "Change Password" option has been moved to the top of Personal Info settings, right above Preferences — easier to find.
  • Walking speed for liquid applications now displays in gallons per 1,000 sq ft (GPA) instead of mph — a more meaningful unit for lawn care.
🐛 Bug Fixes
  • Fixed an odd visual gap between the product selector and the Greenkeeper chat header.
  • The product selector in AI chat wasn't working — fixed.
  • Fixed an issue where the Save button was hidden behind the keyboard when editing a saved application plan.
  • Cost efficiency calculations on the Inventory tab were returning incorrect values — now accurate.
  • Fixed a layout issue where product search results were spread out awkwardly in the planning stage search bar.
  • Journal date grouping is now timezone-aware — activities you log today will correctly show as "Today," not "Yesterday."
  • The Growing Degree Days tooltip on the home screen wasn't launching — fixed.
  • The Getting Started wizard no longer flashes briefly at the top of the home screen when the app first opens.
  • Fixed a layout issue on the Profile page where padding was off after a collapse animation.
  • Fixed an issue where the keyboard would cover the equipment input field during the onboarding wizard.
  • Zone name text on the onboarding screen was spreading out incorrectly — now displays properly.
  • Fixed an issue where the splash screen showed faint white lines at the top and bottom while loading.
🔧 Under the Hood
  • Consolidated three separate product form components into one shared form — keeping the codebase consistent and making future improvements easier to ship.
Build 1.0.0 (12)
April 18, 2026 iOS · TestFlight

Focused on photo capture, AI chat improvements, and a round of bug fixes surfaced by early testers.

✨5 Enhancements
⚙️3 UI Improvements
🐛6 Bug Fixes
✨ Enhancements
  • You can now share AI conversations — great for saving advice or passing along lawn care tips.
  • AI responses now display more slowly so you can read along as text streams in, rather than it all appearing at once.
  • Added a photo option to the Application Wizard — you can now document your applications visually as you log them.
  • Activity entries now support multiple photos, giving you a richer record of your lawn's progress over time.
  • Added a "Take Photo" shortcut to the activity photo capture flow for faster in-the-moment logging.
⚙️ UI Improvements
  • Renamed the AI assistant to Greenkeeper — a more fitting name for your personal lawn care advisor.
  • Removed hashtags from AI chat responses for a cleaner, more readable experience.
  • Added a Growing Degree Days (GDD) tooltip — tap to learn what GDD means and how it helps you time lawn care applications.
🐛 Bug Fixes
  • Fixed a visual gap between the product selector and the Greenkeeper chat header.
  • The AI product selector was unresponsive — now works correctly.
  • Fixed the Save button being hidden behind the keyboard when editing saved application plans.
  • Cost efficiency figures on the Inventory tab were calculating incorrectly — resolved.
  • Product search results in the planning stage were spreading out oddly in the search bar — fixed the layout.
  • Fixed an issue where completing the new account setup flow left the dashboard unaware that yard zones, equipment, and products had been added — your data now appears immediately after onboarding.
  • Fixed the equipment input field being covered by the keyboard during the onboarding wizard.
